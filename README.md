# Bootstrap Instagram Clone

## Proje Açıklaması
Bu proje, Bootstrap kullanılarak hazırlanmış bir Instagram clone çalışmasıdır.
Amaç; Bootstrap grid sistemi, utility class’lar ve ek CSS düzenlemeleri ile
Instagram arayüzüne benzer bir yapı oluşturmaktır.

Çalışma, Kodluyoruz tarafından verilen “broken instagram” şablonu üzerinden
düzenlenerek geliştirilmiştir.

---

## Dosya Yapısı
Proje aşağıdaki dosya ve klasörlerden oluşmaktadır:

- index.html → Ana HTML dosyası
- style.css → Projeye ait özel CSS dosyası
- assets/
  - search.svg → Arama inputu için kullanılan ikon

Tüm dosyalar aynı ana klasör içerisinde yer almaktadır.

---

## Kullanılan Teknolojiler
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome (ikonlar için)

Harici JavaScript yazılmamıştır.

---

## Yapılan Düzenlemeler ve Özellikler
- Navbar sabitlenmiştir (`fixed-top`)
- Navbar yüksekliği 54px olarak ayarlanmıştır
- Body’ye navbar yüksekliği kadar padding eklenmiştir
- Navbar arka planı beyaz yapılmıştır
- Logo sola doğru 192px margin ile konumlandırılmıştır
- Arama alanı `d-flex` ile ortalanmıştır
- Arama inputuna arama ikonu eklenmiştir
- Sağ üst menüye margin verilmiş ve küçük ekranlarda gizlenmiştir
- Orta içerik alanı Bootstrap grid sistemi ile düzenlenmiştir
- `middlearea` için maksimum yükseklik 200px olarak belirlenmiştir
- Hikaye isimleri görsellerin altına hizalanmıştır
- Üç nokta (options) ikonu sağa hizalanmıştır
- Beğenme, yorum ve paylaşma alanlarında border kaldırılmıştır
- Bookmark ikonu sağa hizalanmıştır
- Card header ve footer arka planları beyaz yapılmıştır
- Yorum paylaş metni sağa hizalanmıştır
- Sağ panel sticky hale getirilmiştir
- Sağ panel beyaz arka planlı ve kenarlıksızdır
- Sayfa arka plan rengi Instagram’a uygun olacak şekilde ayarlanmıştır

---

## Projenin Çalıştırılması
1. ZIP dosyası çıkartılır.
2. index.html, style.css ve assets klasörünün aynı dizinde olduğu kontrol edilir.
3. index.html dosyası çift tıklanarak tarayıcıda açılır.
4. Proje Bootstrap ve Font Awesome CDN’leri üzerinden çalışmaktadır, ek kurulum gerekmez.
